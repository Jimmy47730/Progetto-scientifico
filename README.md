# ZeroMirror

Questo repository contiene il sito web e i materiali di presentazione per un progetto scientifico di maturità incentrato sullo studio di un fascio di onde sonore parallelo che attraversa metamateriali acustici.

## Contenuti principali
- static/html/: pagine web del sito di presentazione del progetto.
- static/Paper scientifico - Fresnel mirror.txt: documento tecnico che descrive il modello matematico, le simulazioni e le misure sperimentali.
- audio_comparator/: package Python progettato per analizzare l'input di due microfoni e confrontare i segnali acquisiti.

## Strumenti e tool
Il progetto prevede una suite di strumenti software destinati all'analisi acustica:
- audio_comparator: confronto di due canali microfonici per evidenziare variazioni nella forma d'onda e nello spettro.
- altri tool simili saranno sviluppati per funzioni complementari, come l'analisi del fascio e la progettazione dei metamateriali.

## Obiettivo
L'obiettivo è documentare e presentare un workflow completo: dalla modellazione del fascio sonoro parallelo, alla realizzazione sperimentale, fino all'analisi dei metamateriali e alla fruizione dei tool software.

## Note
Il sito e i materiali contenuti in questo repository sono pensati come supporto al progetto e alla sua dimostrazione, oltre che come punto di partenza per la pubblicazione dei tool associati su repository GitHub separate.
